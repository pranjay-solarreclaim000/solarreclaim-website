## STATUS DEFINITIONS

| Status | Meaning |
|--------|---------|
| Prospect | ICP verified, outreach started |
| Discovery Call Booked | Call scheduled |
| Discovery Call Done | Call completed, proposal pending |
| Agreement Sent | Awaiting signature |
| Active Client | Signed and running campaigns |
| Stalled | No response after follow-up sequence |
| Closed Lost | Declined or disqualified |
